## 2. COST CM 기성실사 공통 구조

```
Ⅰ. 사업개요 — 건축개요 (공사금액, 면적, 기간)
Ⅱ. 공정검토
  - 공정율 관리 현황
  - 공정진행현황 (공종별 합계 행)
Ⅲ. 기성확인 및 검토
  - 기성금액 산정 기준 (계약금액)
  - 도급공사 기성 / 기성금액 확정 (누계기성 %)
```

## 3. 추출 필드 매핑

| PDF 섹션 | DB 필드 | 파싱 패턴 |
|----------|---------|-----------|
| 『용역명』 | sites.name | `『...』` |
| 제N회 CM기성실사 | progress_reports.notes | `제\s*(\d+)\s*회` |
| 합계 행 (공정진행현황) | overall/planned_progress_pct | `합\s*계 100% ... 누계` |
| 공종별 행 | progress_reports.details | 토목/건축/기계/... |
| 누계기성 N% | fund_schedules + latestFundPct | `누계기성\s+([\d.]+)%` |
| 공사도급계약금액 | sites.contract_amount | `₩([\d,]+)` |
| 시공사 | sites.contractor | `시공사\s+...` |
